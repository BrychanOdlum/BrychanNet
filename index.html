A text based RPG game, a pet project from when I was 14 up until 16. The game is written in PHP with a CSS & HTML front-end with heavy use of JavaScript. It incorporates a substantial MySQL database enforced with a REDIS cache. The game has seen over 25,000 players register with revenue being in the 5-figure (£ GBP) bracket.

The code for the live site is currently messy, updates which were in the pipeline aimed to release a new version of the game. Losses were cut, new projects were pursued. Unreleased versions used Node.JS, AngularJS and CSS3.

MCHero

Another 'bedroom project', MCHero was a game server hosting business. The business hosted hundreds of virtual game servers for customers.

MCHero was built using a custom implementation of a game control panel, the WHMCS billing panel, and custom theming. All design was my responsibility, I headed advertising, with other staff maintaining support channels at times.

The site was built to be versatile and responsive, with SSL and DDoS protection enforcing this. The site template was completely custom with all PHP, HTML, CSS, and JavaScript being created and maintained by myself using the Twig template engine.
